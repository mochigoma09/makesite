ℹ️ 各種APIキーの取得方法（クリックして展開）

🔑 1. Gemini APIキー

  1. Google AI Studio で作成してコピー。

🐙 2. GitHub PAT (Personal Access Token)

  1. GitHub トークン作成(Classic) で repo にチェックを入れて作成。

📷 3. Unsplash Access Key (画像用)

  1. Unsplashで無料アカウントを作成し、メール認証を済ませます。
  2. ログインした状態でDevelopersページにアクセスし、右上の「Your apps」をクリックします。
  3. 「New Application」をクリックし、表示される規約のチェックボックス全てにチェックを入れて「Accept terms」を押します。
  4. アプリ名(適当でOK)と説明を入力して「Create application」を押します。
  5. 画面を少し下にスクロールし、Keysセクションにある「Access Key」をコピーします。

⚙️ 基本設定（必須）

🖼️ 画像自動取得 任意

Unsplashのキーを入力すると、AIが要望から自動で最適な画像を検索して組み込みます。
